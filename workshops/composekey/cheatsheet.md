# Living With Linux V supplement: Compose Key Cheat Sheet
This is a list of characters you can type using the Compose key and their respective key combinations. It's not an exhaustive or complete list, but it should be enough to get you through everyday tasks. Some apps might not let you type these characters, such as the up/down arrows and emojis.
## Letters
>[!TIP]
>To type uppercase variants of these characters, type the base character in uppercase.

>[!TIP]
>The last character in the above table, ⓐ, can be made to work with any letter or number, uppercase or lowercase. Simply replace the `a` in the middle with your character. You can also repalce the `a` with a `10`, resulting in a ⑩.
